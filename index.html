🎹 Cours de Piano Gospel

Maîtrisez l'art du piano gospel avec style et spiritualité

🎵Module 1: Fondamentaux du Gospel

🎼 Histoire et Caractéristiques

Le piano gospel trouve ses racines dans la musique spirituelle afro-américaine. Caractéristiques principales :

  • Utilisation intensive d'accords étendus (7e, 9e, 11e)
  • Syncopes et rythmes complexes
  • Main gauche rythmique puissante
  • Ornementation mélodique riche

Accords de base en Do majeur :

I    - C maj7  : C - E - G - B
ii   - D min7  : D - F - A - C
iii  - E min7  : E - G - B - D
IV   - F maj7  : F - A - C - E
V    - G dom7  : G - B - D - F
vi   - A min7  : A - C - E - G
vii  - B ø7    : B - D - F - A

🎯Module 2: Accords et Voicings

🎼 Accords Étendus

Le gospel utilise massivement les extensions d'accords :

  • Maj7 : Sonorité douce et jazzy
  • Dom7 : Tension qui appelle résolution
  • Min7 : Couleur mélancolique
  • Add9 : Ajout de couleur sans tension

Extensions courantes :

C maj9  : C - E - G - B - D
G13     : G - B - D - F - A - C - E
F maj7#11: F - A - C - E - B
A min11 : A - C - E - G - B - D

🎵 Voicings Gospel Authentiques

Technique du "Shell Voicing"

Main gauche: 3e + 7e de l'accord

Main droite: Fondamentale + 5e + extensions

Exemple sur C maj7 :
MG: E + B
MD: C + G + D (add9)

  • Swing : Division ternaire du temps
  • Straight : Division binaire avec syncopes

🎨Module 4: Techniques d'Ornementation

🎵 Ornementations Gospel

Les techniques d'ornementation donnent vie au gospel :

  • Runs : Passages rapides et fluides
  • Fills : Remplissages entre phrases
  • Trills : Alternance rapide de notes
  • Grace notes : Notes d'ornement brèves

🏃 Gospel Runs Essentiels

Run 1 - Gamme pentatonique

Main Droite (Do majeur) :
C - D - E - G - A - C - A - G - E - D - C
(1 - 2 - 3 - 5 - 6 - 8 - 6 - 5 - 3 - 2 - 1)

Run 2 - Chromatique descendant

Main Droite :
C - B - Bb - A - Ab - G - Gb - F - E - Eb - D - C

🎶 Fills Gospel Classiques

Fill de fin de phrase

Temps 3.5-4 d'une mesure :
MD: G - A - B - C (rapide)
MG: C maj7 (tenu)

Fill de transition

Entre deux accords :
C vers F : MD: E - F - G - A
          MG: C - F

🎹Module 5: Styles Gospel Avancés

🏛️ Gospel Traditionnel

Caractéristiques du style traditionnel :

  • Tempo modéré (80-120 BPM)
  • Accords simples avec extensions
  • Main gauche "stride"
  • Mélodie lyrique et expressive

Progression Typique

I - IV - I - V7 - I

C - F - C - G7 - C

🆕 Gospel Contemporain

Style plus moderne avec influences jazz et R&B :

  • Harmonie complexe (extensions, substitutions)
  • Rythmes plus variés
  • Utilisation de modes
  • Techniques de réharmonisation

Réharmonisation I-vi-IV-V

Basique : C - Am - F - G
Avancé  : C maj9 - Am11 - F maj7#11 - G13sus4-G13

🏙️ Urban Gospel

Fusion gospel-hip hop-R&B :

  • Rythmes trap et hip-hop
  • Basses lourdes et percussives
  • Utilisation de samples
  • Harmonie moderne et audacieuse

🎼Module 6: Improvisation et Performance

🎭 Techniques d'Improvisation

L'improvisation gospel repose sur :

  • Connaissance des gammes : Pentatonique, blues, modes
  • Phrases mélodiques : Licks et patterns préparés
  • Écoute active : Réagir aux autres musiciens
  • Expression émotionnelle : Transmettre le message

Gammes pour improvisation en Do

Pentatonique majeure : C - D - E - G - A
Pentatonique mineure : C - Eb - F - G - Bb
Blues                : C - Eb - F - Gb - G - Bb
Dorien              : C - D - Eb - F - G - A - Bb

🎤 Conseils de Performance

  • Communication : Établir le contact avec l'audience
  • Dynamiques : Varier l'intensité selon le message
  • Espace : Laisser respirer la musique
  • Spontanéité : Être prêt à s'adapter

Structure de Performance Gospel

1. Introduction calme

2. Premier couplet (simple)

3. Refrain (plus intense)

4. Deuxième couplet (orné)

5. Refrain final (climax)

6. Outro (retour au calme)
